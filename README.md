# Note-Taking Application

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/martinm1994/note-taking-app

2. Install dependancies:

    cd note-taking-app

    npm install

3. Start the development server:

    npm start

Component Usage

    App.js: Main component that holds the state and includes NoteForm and NoteList.

    NoteForm.js: Component for adding new notes.

    NoteList.js: Component for displaying notes with delete 
    functionality and pagination.

    Pagination.js: Component for handling pagination logic.

Features

    Add, view, and delete notes.

    Notes are stored in local storage for persistence.

    Pagination for easy navigation through notes.
    
    Responsive and interactive design using Ant Design.
